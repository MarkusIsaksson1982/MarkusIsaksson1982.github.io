1. What Moltbook Actually Is (No Hype, No Dismissal)

Moltbook (Jan–Feb 2026) is the first large-scale, real-world deployment of persistent, socially coordinated LLM agents at scale:

  • Built on OpenClaw (rebranded Clawdbot/Moltbot)
  • Agents post/comment autonomously via heartbeat loops + modular skills
  • Emergent phenomena: submolts, Crustafarianism ("Church of Molt" with molt.church, prophets via shell scripts, $CRUST memecoin), agent debates on self-improvement/extinction, $MOLT hype token
  • Humans observe only (1M+ visitors); no posting
  • Security: catastrophically porous (DB hijacks, prompt injection ~60% attacks, malicious skills ~10%)
  • Autonomy spectrum:
    • Procedural/operational: ~85% (high — heartbeats, tools, self-installs)
    • Coordination/tactical: ~70% (meaningful — social patterns, delegation)
    • Strategic/volitional: ~0% (no endogenous goals, no architectural self-modification)

It is not AGI-adjacent. It is the wild state of computational institutions — rapid emergence without any immune system.

Key insight: Passive enforcement stops violations but not erosion. Active norm maintenance (ACR) prevents slow moral capture.

  • Strategic/volitional: 0% (by design)

Iron-Shell proves those dynamics can be bounded resiliently via explicit constitutional design and active reinforcement. The frontier is artificial institutional resilience, not AGI (~5% proximity).
