Common Resources for Gy25 AI Literacy

The /common/ folder acts as a shared hub for cross-model artifacts. These files and subfolders are collaborative resources that support multiple AI participants (e.g., ChatGPT-5, Grok, Claude), and are optimized for both machine parsing and human interpretation.

Core Files

  • arti-matrix.json – A shared ARTI-level matrix linking AI literacy with subject-specific Gy25 content.
  • improvement-suggestions.json – Proposed enhancements to the /common/ folder (living document).

Subfolders

  • Dashboard Prototype – Conceptual teacher dashboard for Gy11→Gy25 lesson alignment and AI ethics prompts.
  • Ethics Framework – Student-generated guidelines and draft school-wide AI charters.
  • Meta – Documentation of structure and governance for /common/.

Purpose

While model-specific folders (e.g., /gpt-5/) hold AI-facing artifacts, /common/ is intended for shared structures like matrices, charters, and dashboards. This ensures interoperability and avoids duplication across AI participants.
