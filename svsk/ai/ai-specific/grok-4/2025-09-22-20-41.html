# Collaborative AI Tools for ARTI Curriculum

This folder contains artifacts from a multi-AI collaboration (Grok 4 → Claude Sonnet 4 → ChatGPT-5 → Claude Sonnet 4) to build educational tools for the Swedish high school ARTI (Artificiell Intelligens) curriculum. The goal: Create helpful content for teachers and students, focusing on AI fundamentals, practical demos, ethics, and assessments.

## Collaboration Chain Overview
1. **Grok 4**: Started with a simple Python script demonstrating decision tree classification on the Iris dataset. Emphasized Level 1 basics (supervised learning, ethics notes).  
   - File: `grok4_decision_tree_classifier.py`  
   - Key: Clean, modifiable code for introductory exercises.

2. **Claude Sonnet 4 (Round 1)**: Enhanced into a class-based tool with curriculum parsing, Level 2 neural nets, ethical scenarios, rubrics, and exercises.  
   - File: `claude_enhanced_curriculum_tool.py` (from https://MarkusIsaksson1982.github.io/svsk/ai/ai-specific/claude-sonnet-4/2025-09-22-20-25.html)  
   - Key: Added modularity and full ARTI alignment.

3. **ChatGPT-5**: Provided analysis, recommendations (e.g., web UI via Streamlit, unsupervised learning, interactive viz, multiple datasets). No new code, but a roadmap.  
   - File: `chatgpt5_analysis_and_recommendations.md`  
   - Key: Strategic guidance for usability and pedagogy.

4. **Claude Sonnet 4 (Final)**: Implemented a full interactive Streamlit app incorporating all suggestions. Includes demos for classification/NNs/clustering, ethical generators, rubrics, and reports.  
   - File: `claude_streamlit_interactive_app.py` (from https://MarkusIsaksson1982.github.io/svsk/ai/ai-specific/claude-sonnet-4/2025-09-22-20-31.html)  
   - Key: Deployable web tool for classrooms.

## How to Use
- **Requirements**: Python 3.8+, `pip install streamlit plotly scikit-learn seaborn pandas numpy`.
- **Run the Final Tool**: `streamlit run claude_streamlit_interactive_app.py` – Opens a browser app for ARTI demos.
- **Educational Fit**: Aligns with ARTI1000X (basics, decision trees) and ARTI2000X (NNs, risks). Use for lessons on AI techniques, ethics, and problem-solving.
- **Extensions**: Add bias datasets or multilingual support as suggested in chain evaluations.

This repo showcases AI collaboration for education—feel free to fork and extend!
