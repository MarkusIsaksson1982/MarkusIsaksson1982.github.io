Interactive Assessment Rubric Component Library

Description

Comprehensive React component library for Swedish gymnasium teacher assessment and scaffolding tools

Prompt

Develop a React component library for creating interactive assessment rubrics and scaffolding tools for Swedish gymnasium teachers. Include components for: dynamic rubric builders with customizable criteria and progression levels, student self-assessment sliders with confidence ratings, peer feedback collection systems, and portfolio evidence trackers. The library should support offline functionality for schools with limited internet, export assessment data to Excel/Google Sheets formats, and include accessibility features following WCAG guidelines. Add multilingual support for Swedish/English interfaces and customizable themes that align with different school branding requirements.

Component Specifications

Dynamic Rubric Builder

  • Customizable criteria and progression levels
  • Dynamic scaling for different assessment types
  • Teacher and administrator interfaces

Student Self-Assessment

  • Confidence rating sliders
  • Progress tracking capabilities
  • Reflection prompt integration

Peer Feedback System

  • Structured feedback collection forms
  • Anonymity options for safe feedback
  • Teacher moderation tools

Portfolio Evidence Tracker

  • Evidence upload and organization
  • Competency mapping features
  • Visual progress tracking

Technical Features

  • Offline Functionality: Works without internet connectivity
  • Export Capabilities: Excel and Google Sheets integration
  • Accessibility: WCAG compliant design
  • Multilingual: Swedish and English support
  • Customizable Themes: School branding alignment
