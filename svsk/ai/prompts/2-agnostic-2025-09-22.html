Advanced Swedish Curriculum Parser with Multi-format Output

Description

Comprehensive Python script for parsing Skolverket ARTI syllabi with automatic subject bridge generation

Prompt

Write Python code using PyMuPDF or pdfplumber to extract and parse Swedish curriculum documents (like Skolverket ARTI syllabi). The script should identify and categorize kunskapskrav sections by grade levels (E/C/A), extract key competency descriptors, and automatically generate potential bridge connections to other subjects (Svenska, Historia, Samhällskunskap) based on keyword matching. Output structured data in multiple formats: CSV for spreadsheet import, JSON for web applications, and markdown for documentation. Include error handling for malformed PDFs, support for both Swedish and English text extraction, and a configuration file system for customizing subject bridge mappings.

Technical Requirements

  • Language: Python
  • Libraries: PyMuPDF, pdfplumber, json, csv, configparser
  • Output Formats: CSV, JSON, Markdown
  • Features: Error handling, multilingual support, keyword matching, configuration system

Expected Outputs

  • CSV: Spreadsheet-compatible grade requirements
  • JSON: Structured data for web applications
  • Markdown: Human-readable documentation
  • Config: Customizable subject bridge mappings
